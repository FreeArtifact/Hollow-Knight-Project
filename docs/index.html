What is this Project?

This project focuses on the characters in the 2017 game Hollow Knight. Hollow Knight is a 2D open-world action-adventure game that primarily takes place inside of Hallownest: a beautiful underground world home to all kinds of characters and experiences.

This project began as a more isolated study of the Hunter's journal and expanded from there. Although there are a number of wonderful websites and wikis dedicated to Hollow Knight, I wanted to create an experience dedicated primarily to the bugs of Hallownest while studying the language and dialogue featured in the game.
